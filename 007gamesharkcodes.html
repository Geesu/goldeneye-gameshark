No Clipping Codes

For those that don't know No Clipping codes are what allow you to walk through walls and go through other boundaries you can't normally.
         Note: Only use one no clipping code at a time.
         After you begin the level that you entered the code for Press L + R together you will
         then be moved to a new position. To turn the code off, you need to go to a part of the level
         that is at the same height as the place you first pressed L + R, and is not in a wall, on water, etc.
         If you ever get stuck press the Gameshark Button and you will be moved to a new position.
